# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Suspected vulnerabilities must immediately be reported, and anyone caught leaking content to the public before any scheduled release will have permissions permenantly removed, be blacklisted, and banned from all moderated platforms.
